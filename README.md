## About 
This program calculates the following:

- Population Mean
- Median
- Mode
- Population Standard Deviation
- Variance of population proportion
- Z-Score
- Standardized score
- Population Correlation Coefficient
- Confidence Interval
- Population Variance
- P Value
- Proportion
- Sample Mean
- Sample Standard Deviation
- Variance of sample proportion
